# 📊 Sistema Inteligente de Previsão de Falhas no Atendimento

## 🎯 Objetivo
Prever falhas operacionais em processos de atendimento utilizando Machine Learning e Lean Six Sigma.

---

## 🧠 Problema
Baixo nível de serviço (NS) impactando desempenho operacional e financeiro.

---

## 📈 Resultados

- Acurácia do modelo: 91%
- Recall (falhas): 96%
- Ganho estimado: R$ 378 mil/ano

---

## 🧪 Técnicas utilizadas

- Análise exploratória de dados (EDA)
- Testes estatísticos (ANOVA, teste t)
- Machine Learning (Random Forest)
- Feature Engineering
- Ajuste de threshold

---

## 🤖 Aplicação

Sistema web interativo desenvolvido com Streamlit:

- Previsão de falha
- Simulação de cenário
- Cálculo de impacto financeiro
- Recomendações automáticas
